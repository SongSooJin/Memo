자바수업

1.변수 : 상태를 저장하는 박스, 변수는 박스를 가리키는 참조(태그)
2.메소드 : 기능을 저장하는 박스,변수를 조작,조건문(if)으로 처리를 분기

3.클래스 : 다수의 변수,메소드를 저장하는 박스
자바 프로그래밍에서 사용하는 기본 빌딩 블록이다.
클래스는 캡슐화 지원한다.

4.패키지 : 
물리적으로 다수의 소스코드인 클래스 (~.java, ~.class)를 저장하는 폴더이며
논리적으로 다수의 클래스를  유니크하게 분류하는 방법이다.
패키지켱은 계층적 부모-자식관계가 없다.
com.google 패키지와 com.google.jcom 패키지는 코드적으로 관련이 업는 패키지이다.
