%d : Integer, Long 타입의 수를 10진 정수로 출력한다. char는 8bit 정수이므로 출력할 수 있다.
%x : Unsigned integer 타입의 수를 16진수로 출력한다. 소문자로 출력한다.
%X : %x와 같은데 대문자로 출력한다.
%#x : %x와 같은데 앞에 0x를 추가해서 16진수임을 명시한다.
%c : 1개의 Character를 출력한다. 물론 char는 8bit 정수이므로 int형 수 중 0~255사이의 수는 출력가능하다. 'a'와 같은 표현을 사용하면 글자를 쉽게 출력할 수 있다.
%s : 1개의 String(Character의 배열)을 출력한다. "Hello World!" 같은 경우 여러 개의 character로 이루어진 string이다. "을 사용해서 나타낸다.
%f : 1개의 float형 타입의 수를 출력한다.
%lf : (Long Float) 1개의 double형 타입의 수를 출력한다. (LF이다.)

1번과 2번식은 같은 값을 갖는다.

1 : for (cnt = 1; cnt <= 10; cnt++) {
			printf("안녕하세요 cnt=%d\n", cnt);
		}

			printf("안녕하세요 cnt=%d\n", cnt);
		}
